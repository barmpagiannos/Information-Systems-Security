# Ασφάλεια Πληροφοριακών Συστημάτων - Password Manager (Passman)

Αυτό το αποθετήριο περιέχει τον πηγαίο κώδικα για την εργασία του μαθήματος **«Ασφάλεια Πληροφοριακών Συστημάτων»** (Ακαδημαϊκό Έτος 2025-2026) του Τμήματος Ηλεκτρολόγων Μηχανικών και Μηχανικών Υπολογιστών (ΑΠΘ).

Σκοπός της εργασίας είναι ο εντοπισμός κρίσιμων κενών ασφαλείας σε μια web εφαρμογή διαχείρισης κωδικών πρόσβασης και η θωράκισή της με τη χρήση σύγχρονων κρυπτογραφικών μεθόδων και βέλτιστων πρακτικών προγραμματισμού.

## Τεχνολογίες
* **Backend:** PHP
* **Database:** MySQL
* **Frontend:** HTML5 / CSS3
* **Security:** OpenSSL (AES-256-GCM), Salted Hashing (SHA-256)

## Βελτιώσεις Ασφαλείας
Η αρχική εφαρμογή παρουσίαζε ευπάθειες που διορθώθηκαν πλήρως:

1. **Προστασία από SQL Injection:** * Αντικατάσταση των απλών ερωτημάτων με **Prepared Statements** και **Parameterized Queries** (`$stmt->prepare()`, `$stmt->bind_param()`).
2. **Θωράκιση από Cross-Site Scripting (XSS):**
   * Εφαρμογή **Output Encoding** σε όλα τα δυναμικά δεδομένα πριν την προβολή τους, με τη χρήση της συνάρτησης `htmlspecialchars()` (ENT_QUOTES, UTF-8).
3. **Ασφαλής Διαχείριση Συνεδριών (Sessions):**
   * Χρήση `session_regenerate_id(true)` κατά τη σύνδεση για αποτροπή επιθέσεων **Session Fixation**.
   * Έλεγχοι ταυτοποίησης σε κάθε σελίδα (`dashboard.php`, `notes.php`).
4. **Ισχυρή Κρυπτογραφία & Hashing:**
   * Αποθήκευση κωδικών χρήστη με **Salted Hashing (SHA-256)**.
   * Κρυπτογράφηση ευαίσθητων δεδομένων (Passwords/Notes) με το πρότυπο **AES-256-GCM**, το οποίο διασφαλίζει τόσο την εμπιστευτικότητα όσο και την ακεραιότητα (Authenticated Encryption).

## Δομή Αρχείων
* `index.html`: Η κεντρική σελίδα υποδοχής.
* `register.php`: Ασφαλής εγγραφή νέου χρήστη με hashing κωδικού.
* `login.php` / `logout.php`: Διαδικασίες σύνδεσης και αποσύνδεσης με αναγέννηση session.
* `dashboard.php`: Διαχείριση και εμφάνιση αποθηκευμένων διαπιστευτηρίων ιστοσελίδων.
* `notes.php`: Σελίδα ανακοινώσεων με προστασία από XSS injection.
* `test_hash.php`: Παραδείγματα υλοποίησης Salted Hashing.
* `test_encrypt.php`: Παραδείγματα κρυπτογράφησης/αποκρυπτογράφησης AES-256-GCM.

## Οδηγίες Εγκατάστασης
1. Αντιγράψτε τα αρχεία στον φάκελο του web server σας (π.χ. `htdocs` για XAMPP).
2. Δημιουργήστε τη βάση δεδομένων `pwd_mgr` στη MySQL.
3. Ρυθμίστε τα στοιχεία σύνδεσης στο αρχείο `db_connect.php`.
4. Πλοηγηθείτε στο `http://localhost/passman/index.html`.
